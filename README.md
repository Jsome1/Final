# Final Project – Responsive Guitar Store

ეს არის პერსონალური ვებსაიტი გიტარის მაღაზიისთვის, შექმნილი HTML და CSS-ის გამოყენებით. საიტი ოპტიმიზებულია სხვადასხვა მოწყობილობაზე სანახავად, რისთვისაც გათვალისწინებულია responsive დიზაინის პრინციპები.

---

## 📄 გვერდები და ფუნქციონალი

**Home:** მთავარი გვერდი, სადაც წარმოდგენილია **header**, **home-banner**, **most popular** სექციები და **footer**. Header-ში არის ლოგო და სანავიგაციო მენიუ. Most popular სექციაში წარმოდგენილია პროდუქტების ბარათები. Footer ცენტრალურად განთავსებულია გვერდის ბოლოში.

**Catalog:** კატალოგის გვერდი, სადაც ნაჩვენებია გიტარების ბარათები. გამოყენებულია grid layout. არის ფილტრები, sort და navigation ბატონები.

**Contact:** კონტაქტის გვერდი, სადაც წარმოდგენილია ფორმა ელფოსტის გასაგზავნად, საკონტაქტო ინფორმაცია და რუკა. სექცია ცენტრალურად არის მოთავსებული.

---

## 🛠️ გამოყენებული ტექნოლოგიები

- HTML5
- CSS3
- Flexbox / Grid
- Media Queries (რესპონსიულობისთვის)
- Animation (keyframes)
- Google Fonts – Inter
- Font Awesome (სოციალური ბმულებისთვის)
